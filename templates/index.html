MalSafe performs signature-based analysis as well as analyzes the meta-information to produce efficient results. We use the power of machine learning to gather information and detect unknown malware samples.

Key Features

Static Analysis

Static analysis examines a malware file without actually running the program. This is the safest way to analyze as executing the code could infect the system. It views information from malware without even viewing the code.

Machine Learning

Malware analysis provides information about the properties, purpose and behavior of a given piece of software. Machine learning boosts this process using various kinds of data on host and network anti-malware components.

URL Phishing Analysis

URL analysis is a thorough examination of a web address to determine the risk of malicious intent upon visiting the site. This process includes checking of reputation score and IP information of the web address.
